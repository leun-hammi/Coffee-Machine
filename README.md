☕ Coffee Machine Simulator (Python)


A command-line coffee machine program that simulates ordering drinks, handling payments, and managing resources like a real café system.

🚀 Features...
Choose drinks: espresso, latte, cappuccino
Accepts coin-based payment system
Checks resource availability (water, milk, coffee)
Gives change if overpaid
Tracks total revenue
Report & shutdown options

💻 Tech Used...
Python 3

🧠 What I Learned...
Dictionaries & nested data structures
Functions & loops
Condition handling
Real-world logic simulation
Basic system design thinking

📌 Controls...
espresso / latte / cappuccino → order drink
report → check resources
off → shut down machine
